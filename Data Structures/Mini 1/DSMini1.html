DS Mini-Assignment #1

Change the first class's exercise to store the information in a simple linked list instead of an array.

Requirements

In main, instead of an array of 10 structs, declare a head pointer to a linked list. The linked list node should have two fields: a struct containing the SongInfo information and a pointer to the next node in the linked list.

The rest of the program is the same, except that you are replacing the array with the linked list.

You should comment as normal.

Submitting

Call your primary source file mini1.c or mini1.cpp.  Put your linked list code into linkedList.c or linkedList.cpp (or linkedList.h if you use a class).  Put any other code into your already-existing mini1.c or mini1.cpp. You can create any .h files you need in order to support this.  Call your project mini1.exe.

Hand in your source files and .x (renamed .exe) file into the Mini 1 Dropbox once done.

This is not a regular Mini-Assignment, as it is due at 11:59 p.m. on Wednesday.
